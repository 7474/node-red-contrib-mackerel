# node-red-contrib-mackerel

This node is for calling [Mackerel API](https://mackerel.io/ja/api-docs/).

I am referring to the [http request node](https://github.com/spawnrider/node-red-contrib-http-request).

Icon was taken from a https://mackerel.io/ja/brand-assets/.
